# Sandra Modas

## Integrantes da dupla
- Eduardo Gonçalves
- José Matheus

## Descrição do caso escolhido
Site desenvolvido para a Sandra Modas, uma banca de moda feminina localizada na Feira Central de Ceilândia, Brasília-DF, fundada por Marly Sandra.

## Necessidade identificada
A loja não possuía presença digital. A ausência de um site dificultava que novos clientes conhecessem os produtos, o horário de atendimento e a localização da banca.

## Público-alvo
Mulheres de 15 a 80 anos moradoras de Ceilândia e região, que buscam moda feminina acessível e de qualidade.

## Objetivo do site
Apresentar a loja, seus produtos e facilitar o contato com clientes novos e antigos.

## Processo de desenvolvimento
A dupla iniciou identificando as informações mais importantes para o público-alvo: história da loja, produtos e localização. Em seguida, estruturamos o HTML com elementos semânticos, aplicamos a estilização com CSS externo e implementamos o carrossel de imagens com JavaScript. Por fim, adicionamos um botão de contato via WhatsApp usando Bootstrap.

## Principais decisões do projeto
- Utilizamos `header`, `main`, `section` e `footer` para garantir semântica adequada ao HTML
- A paleta de cores em rosa foi escolhida por combinar com o público-alvo feminino da loja
- O carrossel foi posicionado em destaque para apresentar os produtos visualmente
- Aplicamos `object-fit: cover` nas imagens para manter proporção sem distorção

## Funcionalidade em JavaScript
Implementamos um carrossel automático de imagens na página inicial. Ele avança sozinho a cada 4 segundos e também permite navegação manual pelos botões de seta e pelos pontinhos. A funcionalidade foi escolhida pois permite exibir os produtos da loja de forma dinâmica e atrativa.

## Uso de Bootstrap
Utilizamos o componente de botão do Bootstrap para criar o botão de contato via WhatsApp no rodapé. O Bootstrap foi adequado pois fornece um botão visualmente consistente e acessível sem necessidade de estilização extra.

## Testes realizados
- Responsividade testada pelo DevTools do navegador em telas de 320px, 768px e 1280px
- Links e botões verificados manualmente
- Carrossel testado em diferentes navegadores
- Imagens verificadas quanto ao carregamento correto
- Navegação básica por teclado testada
